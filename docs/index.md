# BSLVC Dashboard Documentation

Welcome to the documentation for the **BSLVC Dashboard** (Bamberg Survey of Language Variation and Change). This interactive web application allows researchers to explore and analyze linguistic variation data from multiple varieties of English.

## Quick Links

- **[Technical Documentation](technical.md)**: Description of all available functions
- **[Live Dashboard](https://bslvc-dashboard.example.com)**: Access the application. As the BSLVC data is yet unpublished, the dashboard is in demo mode (i.e., access is password protected). If you need access to the dashboard, please contact Fabian Vetter.

## Overview

The BSLVC Dashboard provides an interactive interface for exploring grammar variation data across different varieties of English. The dashboard includes:

### âœ¨ Key Features

- **Interactive Visualizations**: All plots are interactive - you can zoom, make custom selections, show/hide groups
- **Flexible Data Selection**: Choose participants by variety, age, gender, and completeness
- **Multiple Analysis Modes**: 
  - Participant similarity analysis using dimensionality reduction
  - Item rating distributions and comparisons
  - Identify most discrimanitve features between groups
- **Advanced Filtering**: Quick selection presets and custom filters
- **Data Export**: Download filtered datasets and high-quality SVG plots

### ðŸ“Š Data Coverage

The dashboard currently includes:
- **Grammar Data**: 338 grammatical features across spoken and written modes
- **Participants**: 10 English varieties including 3 ENL (England, Scotland, US), 4 ESL (Gibraltar, India, Malta, Puerto Rico), and 3 EFL (Germany, Slovenia, Sweden)
- **Twin Items**: Paired spoken/written features for mode comparison
